# Task organizer

* Painkiller app
- Project-wise tasks organized
- Priorities assigned to tasks optionally (High, normal, low)
- AI-integration: Describe the objective and the model makes a todo list of tasks; auto-adds them (can be reviewed by the user): Also add task to project depending on the prompt provided

- Option for a minimal view (Upcoming due tasks only shown| completed tasks hidden)
- Integration to send email reminders? Some other form of reminders?
- Integration to send reminders to others
